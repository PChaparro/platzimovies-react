## Local usage

1. Insall dependencies:

**You can also use npn instead of pnpm**

```bash
pnpm i
```

2. Run

Run **deploy / static preview**: 

```bash
pnpm run build
pnpm run preview
```

Run **development environment**:

```bash
pnpm run dev
```
